# NOTANDOR
## Introduction
This README presents the **"Notandor"** project, a project developed in order to pass the *Game Development* exam of the Bachelor's Degree in Computer Engineering at the University of Florence.

This project was developed by Francesco Bonaiuti (SI 7195553), Tommaso Ciccotti (SI 7196985) and Martina Schirone (SI 7205227) between April and May 2026 and therefore referring to the academic year 2025/2026.


## Game Overview
The game is simple in its core design. It consists of **four levels** in total, each taking place inside a single room. In every room, the only door leading to the next level is blocked by a **giant green spider**.

In front of the spider there is a **lamp**, while the room itself contains different types of blocks that replicate the behavior of logical operators such as **NOT**, **OR**, and **AND**.

The player can also interact with movable blocks and emitter blocks, which generate either green or red laser beams. Additionally, an NPC is present in the room to guide the player through the mechanics of the game, explaining the functionality of the different blocks and providing a tutorial experience.

The lore of the game is introduced through an initial cutscene. The rooms explored by the player are actually different sections of an infected computer system that has been overrun by bugs. 

The giant spiders symbolize these bugs.

In each level, the player must **solve a puzzle** by correctly positioning and interacting with the blocks in order to activate the lamp. 

Once the lamp is lit, the bug is resolved: the spider dies, the door opens, and the player can proceed to the next level. By progressing through the levels, the player gradually fixes the infected computer and restores its functionality.

## Resources Used
The project was entirely developed using the **Unity Game Engine**.
Several free assets and Unity packages were also used during development to support both gameplay and presentation features.
    
    Terminal User Interface Sound Effects Pack LITE Edition -- Used for keyboard clicking and interface sound effects.
    
    Free Ambient \& Lo-Fi Sci-Fi Music Starter Pack -- Used for the game's background music and ambient audio.
    
    Jammo Character -- Used for the player character model.

    Quarter View 3D Action Assets Pack -- Used for the NPC character model.
    
    Sci-Fi Lab Kit -- Used for the environment, buildings, and puzzle block models.
    
    Spider Polygon -- Used for the spider enemy model representing the ``bugs''.
